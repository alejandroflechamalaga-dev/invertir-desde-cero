# Invertir desde cero — Flecha

Guia interactiva y gratuita para empezar a invertir desde cero: inflacion, interes compuesto, comisiones, fondos indexados y un plan de accion en 3 pasos.

Contenido educativo, no asesoramiento financiero.

Web: https://invertir-desde-cero.vercel.app

Creado por Alejandro Flecha.
